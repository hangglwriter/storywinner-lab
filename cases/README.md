# 💼 창업·사업 사례 조사

벤치 대상들의 **수익모델**을 따로 해부 (구독 / 코칭 / 대행 / SaaS / 강의 조합).
여기서 "작가용 ISBN 자동발급" "전자책 표지 AI 생성기" 같은 **사업 아이디어가 자연 발생** → 행글 `ideas/`로 승격.

## 조사 양식
```
---
title: "..."
date: YYYY-MM-DD
---
# 사례명
- 무엇을: / 누구에게: / 수익모델: / 규모(추정):
- 핵심 성공요인 3:
- 한국·민티 적용 가능성:
## → 분배 (특히 💡 사업아이디어)
```

아직 사례 없음 — 첫 사례: Substack 글쓰기 코치 / 국내 지식창업(신사임당류)부터.
