# 📚 책 콘텐츠 마스터 (스토리위너컴퍼니 출판)

> 스토리위너컴퍼니 출판 **60권**의 콘텐츠 제작 현황판. 책마다 **4트랙**을 채운다.
> 유입용·책소개는 **여러 개 가능** → 숫자 = 발행된 글 수. `·`=없음. `⚠️`=동명작가 복수책(글 배정 눈으로 확인 권장).

## 🎬 4트랙
| 트랙 | 뭐냐 | 위치 |
|---|---|---|
| **① 책 갤러리** | 카탈로그 상세 페이지 (기본 등록) | `/winner-writing/books/[slug]` |
| **② 책 소개(+인터뷰)** | 책·작가 깊이 소개, 인터뷰 있으면 함께 | `/content` |
| **③ 유입용** | 검색 키워드 노리는 정보형 (권당 여러 개 OK) | `/content` |
| **④ 쇼츠** | 후킹 영상 | 유튜브/릴스 |

## 📊 현황 요약
- 총 **60권** (모두 ① 갤러리 O)
- ② 책소개 글 **15개** · ③ 유입용 글 **24개** · 글 전무 책 **23권** · ④ 쇼츠 **0**
- ⇒ 최소 남은 것: 책소개 없는 45권 + 쇼츠 60권 (+ 유입용은 얼마든지 추가)

## 🎯 우선순위 (제안)
1. **간판 5권 4트랙 완성** — 친절한 말하기 / 나이 오십 / 상담사 부부 소통 레시피 / 그림책 워크북 / 다시,나
2. **2026 신간** (병원 인테리어·신화 I/II·명상편의점) — 홍보 골든타임
3. **글 하나만 있는 책 → 나머지 트랙 보강**
4. **글 전무 23권 → 유입용부터**
