# 📚 책 콘텐츠 마스터 (스토리위너컴퍼니 출판)

> 스토리위너컴퍼니가 출판사로 낸 책 **60권**의 콘텐츠 제작 현황판.
> 책마다 **4트랙**을 채운다. 상태: `O`=완료 · `·`=미제작. (자동 매칭 초안 — 수정은 이 파일에서)

## 🎬 4트랙 정의
| 트랙 | 뭐냐 | 위치 |
|---|---|---|
| **① 책 갤러리** | 카탈로그 상세 페이지 (기본 등록) | `/winner-writing/books/[slug]` |
| **② 책 소개(+인터뷰)** | 책·작가를 깊이 소개하는 글, 작가 인터뷰 있으면 함께 | `/content` |
| **③ 유입용** | 검색 키워드로 유입 노리는 정보전달형 글 | `/content` |
| **④ 쇼츠** | 후킹 영상 | 유튜브/릴스 |

## 📊 현황 요약
- 총 **60권** (모두 ① 책 갤러리 등록 완료)
- ② 책소개 글: **32권** · ③ 유입용 글: **12권** · 둘 다: **5권** · 글 전무: **21권**
- ④ 쇼츠: **0권** (전무 — 신규 개척)
- ⇒ 남은 총량: 책소개 28권 + 유입용 48권 + 쇼츠 60권

## 🎯 우선순위 (제안)
1. **간판 5권 4트랙 완성** — 친절한 말하기 / 나이 오십 / 상담사 부부 소통 레시피 / 그림책 워크북 / 다시,나
2. **2026 신간** (병원 인테리어·신화 I/II·명상편의점) — 홍보 골든타임
3. **글 하나만 있는 책 → 나머지 트랙 채우기**
4. **글 전무 책 → 유입용부터**

## 🔧 제작 규칙 (책 콘텐츠 레시피)
- **책소개(+인터뷰)**: EPUB/원고 실제 내용 기반(지어내기 금지, 노션·서점 원문 fetch). 인터뷰 있으면 함께. `D:\ebook-project\작업\<작가>\책소개.md` 참고
- **유입용**: 네이버 키워드API로 롱테일 검색량 확인 후 제목(추측 금지). 정보전달형
- **쇼츠**: 후킹 스크립트 + 영상. 책소개 글에서 후킹 포인트 추출
- **이미지**: 책표지 or 목업 OG(`generate_mockup_og.py --upload`). 책(서점)링크는 raw URL로 자동 임베딩
- **채널**: 홈피 `/content` → 네이버 블로그 → 브런치. 발행 후 구글문서 기록
- 관련 메모: `content-image-style` · `book-link-raw-embed` · `feedback_no-fabrication`
