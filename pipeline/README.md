| 유형 | 키워드 소스 | 본문 재료 | 이미지 | 채널 | 핵심 규칙 |
|---|---|---|---|---|---|

| **유튜브 영상글** (AI 활용) | 영상 주제 키워드 | 영상 **핵심메시지(척추)** | 영상캡처(`shorts-capture.py`) | 홈피 → 블로그 → 브런치 풀세트 | 기능나열 X · 유튜브 영상 댓글유도 O |
| **위너책쓰기 SEO** | 책쓰기·출판 검색키워드 | 출판방법 정보 | 자체제작 | 홈피`/content` (→블로그) | 기수모집 폼 CTA · 전환 목적 |

**공통 발행 흐름:** 홈피 글 → `content-republish` 스킬 → 네이버 블로그(일기콘 N) → (브런치) → 구글문서 기록(`diarycon_insert.py`).
**작업 시 펼 단일소스:** 발행 메커니즘 [`study/43`] / 강의후기 [메모 `lecture-post-seo-rules`·`lecture-review-backlog`] / 영상글 [메모 `video-content-core-message`] / 책 [메모 `book-link-raw-embed`·`content-image-style`]
**깊은 리서치·작업노트:** 각 글은 `pipeline/<유형>/<주제>.md`에 저장(키워드 실측·구성안·소스). 시장 단위 조사는 `radar/`·`bench/`·`crawl/`.
