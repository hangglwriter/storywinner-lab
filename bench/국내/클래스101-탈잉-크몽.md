# 클래스101 · 탈잉 · 크몽

> 국내 글쓰기·전자책 **수요 검증 채널**. 여기서 잘 팔리는 클래스/전자책 = 시장이 돈 내는 주제.

## 관찰 포인트 (크롤 대상)
- **크몽 전자책 카테고리 베스트** = 실제 결제 데이터 → 무슨 주제가 팔리는지 직답
- 클래스101·탈잉 글쓰기/전자책 클래스 랭킹 + 후기 수 = 수요 깊이
- 강사들의 **세일즈 카피·커리큘럼 구성** = 우리 위너책쓰기 소개 페이지 벤치

## 뺏어올 것
- 베스트 전자책 주제 → 홈피 콘텐츠/책 소개 글감
- 클래스 커리큘럼 vs 위너책쓰기 7주 차별점 정리 → 소개 페이지 강화
