# 📨 뉴스레터

콘텐츠 발행과 **연계**. 레이더/파이프라인에서 나온 꼭지를 호별로 묶어 발송 + 사이트 콘텐츠로 누적.

## 상태
- 기획안: 행글라이터 기획실 `ideas/`에 2건(글라이드·행글라이터 뉴스레터) → 여기로 통합/링크 예정
- 발송 인프라: 스토리위너 홈피 Resend 도입 시 `/newsletter` 아카이브 연동

## 호별 (`newsletter/호별`)
아직 발송 전. 창간호 후보 꼭지:
- 메인: "평범한 사람도 책 쓸 수 있나요?" (AEO #1)
- 트렌드: 부크크/유페이퍼 자가출판 (레이더 #1)
- 도구: 챗GPT로 책 목차 짜기
