# 📡 트렌드 레이더

책쓰기 · 콘텐츠마케팅 · AI글쓰기 통합 트렌드. **월 1회** 갱신.
`crawl`(수요 신호) + `bench`/`cases`(잘하는 사람) 를 종합 → 맨 아래 **`→ 분배`** 로 각 출구에 라우팅.

## 분배 규칙 (모든 레이더·조사 문서 공통)
문서 끝에 반드시:
```
## → 분배
- 🎬 영상감: ... (행글)
- 📝 블로그감: ... (pipeline/책 or ai)
- 📨 뉴스레터: ... 호 꼭지
- 💡 사업아이디어: ... → 행글 ideas/로 승격
- 🎓 강의: 위너책쓰기 N주차 반영
```
