# 🕷 크롤 대상 정의

수요 신호("뭘 써야 팔리나")를 데이터로 수집. 결과는 `crawl/YYYY-MM-DD-*.md` 스냅샷으로.
이미 만든 도구 재사용: selenium(브런치), yt-dlp, 네이버 검색광고 API.

| 대상 | 신호 | 방법 | 주기 |
|---|---|---|---|
| 크몽 전자책 베스트 | 실판매=수요 | selenium | 격주 |
| 예스24·교보 전자책 베스트 | 시장 베스트 | selenium/크롤 | 격주 |
| 부크크 신간/베스트 | POD 신간 동향 | 크롤 | 월 |
| 텀블벅·와디즈 출판 펀딩 | 검증된 수요 | 크롤 | 월 |
| 브런치 출판/글쓰기 인기글 | 주제 트렌드 | selenium | 격주 |
| 네이버 검색광고 API | 키워드 수요 | API(작동중) | 월 |
| Product Hunt — AI writing | 신규 도구 | 크롤/API | 격주 |
| Substack/Medium top writing | 해외 글쓰기 트렌드 | 크롤 | 월 |

## 스냅샷 양식
```
---
title: "크몽 전자책 베스트 (YYYY-MM-DD)"
date: YYYY-MM-DD
---
# ...
- 순위 | 제목 | 가격 | 신호 메모
## → 분배 (레이더로 올릴 항목)
```
